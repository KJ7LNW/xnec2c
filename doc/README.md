# Documentation Directory

## Reference Materials

This directory contains reference documentation for the NEC-2 file format and implementation.

### NEC-2 Reference Files

**nec2dx.f** - NEC-2 double precision FORTRAN source code
- Core NEC-2 code: US Government work (public domain)
- Timing utilities (lines 8180+): Copyright 1990, 1992 The Regents of the University of California
- Included as reference material for NEC-2 format implementation
- Not built or linked into xnec2c

**nec2prt3.txt** - NEC-2 User's Guide (Part III)
- US Government work (public domain)
- Reference documentation for NEC-2 input format
- Text version derived from original printed manual via OCR

**nec2prt3.pdf** - NEC-2 User's Guide (Part III)
- US Government work (public domain)
- PDF version of the NEC-2 User's Guide

**nec2c.txt** - nec2c documentation
- Documents the C translation of NEC-2
- States: "nec2c is Public Domain, same as the original FORTRAN source"

### License Status

The original NEC-2 code was developed at Lawrence Livermore National Laboratory and is US Government work, placing it in the public domain. Reference materials included here are for documentation purposes only and are not compiled or linked into xnec2c.
